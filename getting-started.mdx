## Create Your First Signature

You begin by visiting the Paperguide website and selecting a signature type. The interface guides you through drawing, typing, or uploading. Preview changes in real-time to ensure it fits your needs.

<Steps>
  <Step title="Choose Signature Type" icon="plus">
    Select from handwritten, typed, or upload options.
    <Callout kind="tip">
      Start with typed for simplicity if you're new.
    </Callout>
  </Step>
  <Step title="Draw or Enter Text" icon="edit">
    For handwritten, use your mouse or touchpad. For typed, input your name and style.
    ```javascript
    // Simulate typed signature data
    const signatureData = {
      type: 'typed',
      text: 'John Doe',
      font: 'Signature Script'
    };
    console.log(signatureData);
    ```
  </Step>
  <Step title="Customize Appearance" icon="settings">
    Adjust size, color, and effects like shadow or outline.
  </Step>
  <Step title="Preview and Download" icon="download">
    View the final signature and export in your preferred format.
  </Step>
</Steps>

## Troubleshooting Common Issues

If the drawing tool lags, ensure your browser is updated. For upload errors, check file size limits under 5MB. Contact support via the help section if problems persist.
