## Advanced Customization

For developers, export with metadata for easy integration. Use the API to automate signature generation based on user input.

<CodeGroup tabs="JavaScript,HTML">
  ````javascript
  // Generate signature programmatically
  const generateSignature = async (text, color) => {
    const response = await fetch('/api/signatures', {
      method: 'POST',
      body: JSON.stringify({ text, color })
    });
    return response.json();
  };
  ````
  ```html
  <!-- Embed customized signature -->
  <img src="/api/signatures/john-doe?color=#3B82F6" 
       alt="Custom Signature" 
       style="width: 200px; opacity: 0.9;">
  ```
</CodeGroup>
